# ELongATE_CAG_sizing
This repository contains code for SPN CAG sizing from snRNA-seq expression values.

In this repository, you will find:

* **ELongATE_env.yml**: a yml file containing package versions for creating a conda environment
* **Models**: folder including *Phase_model.rds* and *CAG_sizing_model.rds* files, that can be imported into R environment and used for HD-Phase-Model predictions
* **snRNAseqCAG_ELongATE_noteBook.md**: a notebook reporting R code for training, testing and applying the HD-Phase-Model to external brain single-nuclei sequencing datasets
